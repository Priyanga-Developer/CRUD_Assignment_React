# React CRUD User Management App

A modern React-based CRUD (Create, Read, Update, Delete) application built with React 19, TypeScript, Material UI, and TanStack Query.

This project demonstrates scalable frontend architecture with extensibility in mind, allowing new form fields to be added with minimal code changes.
